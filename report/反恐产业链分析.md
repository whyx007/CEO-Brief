## 六、附录：查询方法

### 使用的 Cypher 查询

```cypher
-- 1. 发现与反恐相关的节点
MATCH (n) 
WHERE toLower(n.name) CONTAINS toLower('反恐') 
   OR toLower(n.name) CONTAINS toLower('terror')
   OR toLower(n.name) CONTAINS toLower('安防')
   OR toLower(n.name) CONTAINS toLower('公共安全')
   OR toLower(n.name) CONTAINS toLower('国土安全')

-- 2. 查找关联企业及详细信息
MATCH (e:Enterprise)-[:PROVIDES_PRODUCT|HAS_CAPABILITY|SERVES_INDUSTRY
       |APPLIES_TO_SCENARIO|HAS_CUSTOMER]->(n)
WHERE n.name CONTAINS '关键词'
RETURN e, n

-- 3. 获取企业完整画像
MATCH (e:Enterprise {name: '企业名'})-[r]-(n)
RETURN type(r), labels(n), n.name
```
