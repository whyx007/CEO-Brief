# CEO参阅运行规则（2026-04-21 确认版）

## 整体结构
CEO参阅按三部分组织：

1. **时政 / 政策**
   - 每日保底输出
   - 面向 CEO 的公共决策价值
   - 不要求强命中 target

2. **宏观经济 / 产业**
   - 每日保底输出
   - 由两层组成：
     - **泛经济**：利率、汇率、出口、制造业景气、资本市场、国际供应链、大宗商品、重要宏观政策
     - **产业聚焦**：当前阶段重点聚焦 **商业航天**
   - 商业航天产业部分重点关注：融资、发射、卫星互联网、遥感、测运控、星座、产业链上下游、政策与国际竞争格局

3. **目标相关**
   - 必须严格约束
   - 只展示与目标公司、竞对、上下游、目标关键词强相关的内容
   - 宁可少，也不回退展示泛商业新闻

## 天气
- 天气地点默认使用 **目标公司所在地**
- 当前目标公司：`中科天塔`
- 后续应将天气 location 从通用城市改为目标公司所在城市，并支持配置化

## 当前实现约束
- 第 2 部分（宏观经济 / 产业）需要同时覆盖“泛经济”与“商业航天产业聚焦”两层内容
- 商业航天产业部分需要扩充更贴近赛道的来源与 query
- 第 3 部分（目标相关）保持 strict mode
- 前端展示应明确区分三部分，不要混在一起
